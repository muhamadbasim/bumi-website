# Changelog

## 0.2.0 - 2026-07-09

### Added
- Footer links now route to real Bumi pages instead of `#` placeholders.
- Prototype `/privacy` and `/terms` pages to avoid broken legal links.
- Newsletter prototype validation with success/error feedback.
- Capability card CTA links for all 10 capability cards.
- Insight detail placeholder routes and homepage insight rows mapped to specific article URLs.
- Search empty state plus capability/legal entries in the search overlay.
- Descriptive ARIA labels for mega menu triggers.

### Changed
- Disabled Nuxt DevTools for client-facing preview cleanliness.
- Bumped project version to `0.2.0`.

### Verification
- Run `npm run verify` before release/push.
